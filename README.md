# Document-QA-RAG
📄 Document QA Website

A Document Question-Answering (QA) Web Application built using HTML, CSS, JavaScript, and RAG (Retrieval-Augmented Generation) tools. This website allows users to upload documents and ask questions to get answers directly from the document content.

🔹 Features

Upload Documents: Users can upload documents in supported formats (PDF, TXT, DOCX, etc.).

Ask Questions: Ask questions related to the uploaded documents.

RAG-powered Answers: Answers are generated using Retrieval-Augmented Generation techniques to fetch relevant content from the document.

Interactive UI: Responsive and user-friendly interface built with HTML, CSS, and JS.

Client-side Validation: Ensures users can only upload supported document types.

Lightweight and Fast: Minimal dependencies and optimized for speed.

🛠️ Technologies Used

Frontend:

HTML5

CSS3 (Flexbox/Grid for layout)

JavaScript (Vanilla JS for interactivity)

Backend / RAG Tools:
Fast Api ,Langchain

Embedding models for semantic search

Retrieval algorithms to find relevant sections of the document
